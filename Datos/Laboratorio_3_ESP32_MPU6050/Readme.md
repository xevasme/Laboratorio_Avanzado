# Laboratorio 3 – ESP32, MPU6050 y Control de Servomotor

Este repositorio contiene el desarrollo del **Informe de Laboratorio Nº3** de la asignatura *Laboratorio Avanzado I*. El proyecto consiste en la implementación de un sistema embebido basado en el microcontrolador ESP32 para la adquisición de datos de orientación mediante el sensor inercial MPU6050, el control de un servomotor y la visualización de los datos en tiempo real a través de una interfaz web usando comunicación WiFi.

El sistema permite calcular los ángulos de orientación (roll y pitch) a partir de las mediciones del acelerómetro y giroscopio, los cuales se emplean como referencia para el control del servomotor. Adicionalmente, los datos procesados son transmitidos mediante WiFi y visualizados en una página HTML alojada en el propio ESP32.

## Contenido del repositorio
- Código fuente en MicroPython y Arduino IDE.
- Imágenes del montaje experimental.
